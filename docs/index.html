TAS

Session tracking, token budgeting, input telemetry, slash-command skills, and maintenance cadence for any project.

View on GitHub Quick Start
git clone https://github.com/Voxos-ai-Inc/tas.git /tmp/tas && bash /tmp/tas/setup.sh

What You Get

Session Tracking

Every session auto-registered on start, summarized on end. Crashed tabs detected as orphans with resume offered. Tab concurrency tracked.

Token Budgeting

Estimate effort in tokens, not time. Real token counts extracted from transcripts. Per-session cost calculation. Calibration analytics.

Input Telemetry

Every prompt captured automatically. Analyze message volume, complexity, intent distribution, and tab concurrency over time.

10 Slash-Command Skills

Ships with /done, /queue, /preview, /code-audit, /brainstorm, /recover, /attention, /hypothesis, /pentest, /idea-mining. Create your own with /nu.

Maintenance Cadence

Define recurring tasks with cadences (hourly, daily). Overdue tasks surfaced at session start. Your agent becomes an autonomous TPM.

Hypothesis Protocol

Hypothesis → Set A → Change → Set B → Verdict. No vibes-based development. Six artifact profiles for every change type.

Security Posture

Non-negotiable constraints baked into the CLAUDE.md template: secrets management, auth-by-default, input validation, least-privilege, audit logging.

Session Lifecycle

Session start
  │
  ├─ SessionStart hook fires
  │    ├─ session-register.sh → ~/.claude/session-tracking/<id>.json
  │    └─ Tab concurrency event → input-telemetry/concurrency.jsonl
  │
  ├─ Each user message:
  │    └─ input-capture.sh → input-telemetry/raw.jsonl
  │
  ├─ You work, optionally logging tasks:
  │    ├─ task-log.sh estimate → returns task_id
  │    ├─ task-log.sh start   → marks work begun
  │    └─ task-log.sh complete → records actual effort
  │
  ├─ Session ends
  │    ├─ session-end.sh → parses transcript, calculates cost, writes sessions.jsonl
  │    └─ task-check.sh  → nudges once if no tasks logged
  │
  └─ Later: cc-sessions.sh check
       └─ Finds orphaned sessions → claude --resume <id>
